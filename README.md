This project predicts whether a loan application should be approved using machine learning. It analyzes applicant financial data such as income, credit score, and debt-to-income ratio to classify applications as approved or rejected. A Random Forest model is used to improve prediction accuracy and support decision-making in financial institutions.
Features (Example Dataset)
Income
Loan Amount
Credit Score
Debt-to-Income Ratio (DTI)
Loan Approved (Target: Yes/No)
ML Approach
Type: Classification Problem
Models you can use:
Logistic Regression
Decision Tree
Random Forest ✅ (recommended)
